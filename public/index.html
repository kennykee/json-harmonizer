Hotel Data JSON Harmonizer

Note: The schema in this demo is predefined for hotel data in schema.js. You can paste JSON in different formats, and the harmonizer will attempt to map them to the schema.

Harmonized and Filtered JSON

REST API

POST /harmonize
Harmonize and filter hotel data from multiple JSON sources.
Base URL: Domain in this page
Request Body (application/json):
{
  "jsonList": ["<stringified JSON>", ...],
  "ids": "1,2,3",              // optional, comma-separated hotel ids
  "destinationIds": "100,200"  // optional, comma-separated destination ids
}
Response (application/json):
{
  "success": true,
  "data": "[ ...harmonized hotels as JSON... ]"
}
// or on error
{
  "success": false,
  "message": "Error message"
}
Try it with the UI above or use curl / Postman for API integration.
